# Business Workflow — BOZ Verified Fit

## Complete Hiring Flow

```
SALES sends platform link → CLIENT fills 8-step form + uploads JD → AI analyzes
→ RECRUITER reviews, adds notes, approves → ANA adds soft skills evaluation
→ ADMIN manages pipeline (assign candidates, progress stages, upload docs)
→ ADMIN authorizes review link → SALES sends review link to CLIENT
→ CLIENT accepts/rejects candidates → ADMIN sees decisions
```

## Client 8-Step Form
0. Welcome (name, company) → 1. Upload JD (AI analysis) → 2. Role info → 3. Tech stack (pills) → 3.5 AI suggestions (toggle chips) → 4. Must Have / Nice to Have → 5. Additional (english, timezone, certs) → 6. AI generates objective+responsibilities → 7. Review & Submit

## Profile Status: `new → pending_review → pending_soft → in_progress → sourcing → filled (Ready for Client) → closed`

## Pipeline Stages: `data_verification → schedule_ts → ready_ts → schedule_technical → ready_technical → schedule_client → ready_client → client_approval → hired | waiting | discarded`

## Review Link Flow
1. Admin assigns candidates + progresses through stages
2. Admin clicks "Authorize Review" on profile
3. Sales sees it in Delivery tab as "Live"
4. Sales sends review link to client (email or copy)
5. Client opens link → sees candidates with scores, skills, docs → accepts/rejects

## Finance: Before searching, Finance prices each profile (client rate vs rockstar salary). Viability: 💰>40%, ⚠️20-40%, 🔴<20%
